# CodeAlpha AI FAQ Chatbot

## Description
AI FAQ Chatbot is a web-based chatbot developed using Python Flask, HTML, CSS, and JavaScript. It provides automated answers to user queries through an interactive interface.

## Features
- Interactive chatbot interface
- FAQ based responses
- Real-time user interaction
- Simple and user-friendly design

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the application:

python app.py
